# 🎯 Whack-A-Mole Game (Java Swing)

## 📌 Overview

This project is a simple **Whack-A-Mole game** developed using Java Swing. The player must click on randomly appearing moles within a limited time to score points. It demonstrates event handling, GUI design, and real-time interaction in Java.

---

## 🎮 Features

* Interactive GUI using Java Swing
* Random mole appearance on grid
* Score tracking system
* Countdown timer (30 seconds)
* Game over popup with final score
* Background image for better UI

---

## 🛠️ Technologies Used

* Java
* Swing (GUI Framework)
* AWT (Graphics & Events)

## ▶️ How to Run

1. Open the project in an IDE (IntelliJ / VS Code / BlueJ)
2. Make sure the **Images folder is in correct path**
3. Compile all files
4. Run `WhackAMole.java`

---

## 🎯 How to Play

* Click on the mole as soon as it appears
* Each correct click increases your score
* You have **30 seconds** to score maximum points
* Game ends automatically when time runs out

---

## ⚠️ Important Notes

* Ensure image paths are correct (`images/mole1.png`, etc.)
* If images do not load, update the file path accordingly
* Case sensitivity matters (Images vs images)
